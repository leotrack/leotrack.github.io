关于 MOVABLE TYPE 完全手册

这一份手册的雏形是先前我所撰写的 Movable Type 新手手册；在经过了一段时间的实际使用后，许多读者反应了她们所特别需要（而我却没有写到）的信息；同时藉由几堂授课之后，又有许多需要进一步厘清的观念、或更具体的例子，也都加进了这个版本的手册里。

实际上，你不祇可以把这份完全手册视为新手手册的延伸版本，更可以把它视为另一个单独的版本。因为在我原本的写作计画中，就筹画著要连带地把相关的议题一并放进来；像是如何申请动态网域名称服务 (DynDNS) 、如何调校网志组态与设定档 (mt.cfg) 、如何让网志的接口与样式更具一致性、如何规划要在页面上提供的功能、如何让网志更具亲和力、甚至是如何决定网志的出版授权协议等。

这些议题在新手手册中，都因为篇幅的关系而被删略了；不过现在你可以在这里找到。而本书的适用程度则可以说是从新手一路到中、高阶使用者；所以无论你对于 MT 曾经有过多少认识与了解，都还是能从中获得更多有趣的经验才是。

照惯例，这份手册仍旧可以被当作是 Movable Type 使用手册的导读，也可以当作是使用 MT 的入门书。除此之外，本书也会稍微多加提及 Windows 平台下施用的注意事项，来让这套系统能够更加普及。

本手册撰写之时，虽然会一再强调观念上的要点，但会尽量以简明而不累赘的文字来描述；讲解模版的章节中，也尽可能地附上批注，并且使用高反差的颜色来标出最重要的几列；同时在大多数的章节最后一段，也附注使用手册里相关联的章节，让有兴趣的读者可以进一步地察看更多的细节。

各种缩写字(abbr.)或头文字(acronym)首次出现的时候均会加上虚线底线，祇需把鼠标光标移至其上就可以看到更多讯息；图片也都附上了替代(alt)文字以及图片说明，让无法加载图片的读者也能大略知道图片内的重点。本手册也做了便捷键(accesskey)的设计：无论你正在阅读哪一页，祇要按下 Alt-1 （ IE 的使用者还要多按一次 Enter ）就可以回到手册总目录。所有的图片跟条列的步骤也都加上了定位锚(anchor)；图片按照编号都标上了 <a id="img_NN"></a> ，而条列项目或步骤则标上了 <a id="foobar_NN"></a> ，以方便叫用。

手册撰写难免有不完美之处，还请不吝指教；由于手册撰写以及抓图旷日废时，而软体更新版本的速度又相当之快，因此本手册势必难以保证每一张图片都取自最新版的软体画面。最明显的差异之一，恐怕就是部分的页面是以 Phoenix 0.5 开启后抓的图，而另一些页面却是以 Mozilla FireBird 0.6 开启后抓的图。其中 Mozilla FireBird 0.6 正是 Phoenix 改名后的下一版；另外内容里也混用了分别取自 MovableType 2.63 跟 2.64 版的画面，这是因为本手册撰写一半之后， MovableType 才改版，为了使本手册的进度不至于落后太多，凡是使用接口上无太大差异的画面便不重新抓取了。希望读者能够体谅这一点；当然对于有显著差异的部分，则会尽量取用最新版软体的画面。希望这份手册，多少对于新学乍练的读者有所帮助。
