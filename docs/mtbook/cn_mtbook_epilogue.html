结语

这份文件大概到此就告一个段落，然而所有的事物才刚开始而已。我们还没有机会讨论到通知书的运用，还没有深入探讨 Movable Type 预设模版的每一个角落，更没有实际示范以 Movable Type 作为后台，规划出整个机构或组织的公开站台的过程。

除此之外， Movable Type 还有许许多多不同的外挂模块；有的外挂模块是用于模版之中、增加模版弹性的，而另外还有一些外挂模块则专门用来处理文字格式。这些外挂模块我们也都还没有介绍过。

以 Movable Type 的弹性与变化来说，要在一份文件里就囊括所有相关的信息，将会使得这份文件过于厚重而难以阅读；因此在规划这份文件的时候，这些篇章就因此而被删去了。或许有朝一日还会有写给进阶使用者的文件，分门别类地讨论站台规划、研究外挂模块。不过不是现在。

在这些文件还没有诞生之前，正是你自己摸索的绝佳时机。这些东西都不难，祇是需要你去熟悉罢了。而熟悉没有别的途径，多用常用就是了。在此仅先预祝各位年轻的部落客们「格」得愉快！
