附录三 ─ 相关资源

以下这份列表列出了在繁体中文世界中，几个跟 Movable Type 有关的主要资源；你可以透过这些地方去取得与 Movable Type 相关的大量信息，或者是提出问题、与人讨论：

  • MTBook.net ：本书的线上版本位于 http://mtbook.net/ ，放在这个位置的版本将会时时更新、尽可能以最快的速度修正错误与瑕疵，以及补充新信息。
  • MTBook.org ：与本书有关的各种文件几乎都列在 http://mtbook.org/ ，包含了一份相当精简的「 Movable Type 新手手册」以及全部的官方文件译本。
  • 部落格村：由 rex 建立的网志入口站台，位于 http://www.blog.idv.tw/ ，其中整理了一份部落格建站资源（ http://www.blog.idv.tw/blogresource.htm ），收纳了许多人的作品，值得一看。
  • 深入亲和力：由我和 hlb 一起翻译的 Dive Into Accessibility ，位于http://dia.z6i.org ，这份文件相当简单好读，却讲了非常多增加网页亲和力的技巧，甚至还列了不少用于各种网志系统（包括 Movable Type ）的实际范例。事实上许多在此文件内的技巧，都已经被目前版本的 Movable Type 所采用了。
  • 部落格随便祭：这是位于 http://phpbb.elixus.org/ 的讨论区，其中有一区就是在讨论 Movable Type 。许多在台湾活跃的网志作者也都会在此出没。
  • 台湾部落格 B.B.S 讨论群组：这是由台湾部落格所维护的讨论区，位于 http://bbs.twblog.net/ ，其中跟 Movable Type 相关的讨论都在「部落格麻瓜园」里。
  • 旗标出版公司：本文件的纸本由旗标出版，不习惯阅读屏幕的朋友可以考虑购买纸本。介绍本书的页面位于 http://www.flag.com.tw/book_info/F8542.htm 。
