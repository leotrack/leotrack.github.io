作者简介

本文件作者 Jedi ，本名林克寰，目前就读于国立台北护理学院听语障碍科学研究所硕士班，主修听力学。

作者多年来始终对于网络文化抱持著浓厚的兴趣，曾经参与过 GnuPG 、 GPGShell 、 WinPT 、 BkGnuPG 等跟公开金钥加密法有关的软体中文化，同时维护了 GPGZilla 站台，持续地推广网络隐私与身份权的信息。除此之外也参与了 Perl 学习手册第三版、 Perl 网络程序设计、 Absolute BSD 等书的翻译工作，还跟 hlb 一起翻译了深入亲和力 (Dive Into Accessibility) 。

在跟网志有关的部分，作者曾经四处巡回演讲跟 Blog 及 Movable Type 相关的题目，独力翻译完所有的 Movable Type 官方文件，也与 digdog 及 autrijus 一起维护了 Movable Type 的中文化。诸位此刻所阅读的 Movable Type 完全手册也完全出自作者一人之键盘而成。

作者自己的网志位于 http://Jedi.z6i.org/blog/ ，与本书或 Movable Type 有关的信息多半也会在此公布；常常有许多人也会在此提出与 Movable Type 相关的疑惑，也几乎都能在最短的时间内取得回复。若您真的遇到了问题，也欢迎随时前来留言。
