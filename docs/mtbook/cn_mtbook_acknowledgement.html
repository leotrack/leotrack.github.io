致谢

本书得以完成，必须要感谢许多人。

首先要感谢 ilya 率先让我知道有 Blog 这玩意儿，他的锐利目光总是赶在众人之前就先看到未来的趋势；接著要感谢 hcchien 提供了 http://blog.elixus.org/ 上的空间及 Movable Type 帐号，这才让我开始接触到 Movable Type 。对于懒惰成性的他来说，这种际遇简直可以说是可遇不可求的。

跟 DEBBY 一起研究各种 Movable Type 模版技巧的经验可以说是促使我开始翻译及撰写这些文件的出发点，若不是她的关系，恐怕我至今还没有读完官方手册；而 keichii 的提议则是让我决定要附上详尽图解的关键点，若不是他一再怂恿，这本手册将停留在完全没有图解的阶段。

digdog 对于 Movable Type 中文化的热血与贡献功不可没，工具列按钮上的麦金塔字型更是漂亮无比；而 autrijus 总是抽空帮忙检查 Movable Type 的源码瑕疵、帮忙处理本土化兼容性、制作安装档等，更是令人钦佩！

gugod 所撰写的 outline2html 对于我在推广 Movable Type 帮了非常多忙，而一些 CSS 技巧也是跟他讨论之后才有办法想出来的。

acer 、 ipa 、 Schee 、 zonble 与邹景平老师的文章提供了许多历史性及观念性的重要信息， hlb 三不五时就丢出来的有趣链结则拓广了我对于 Movable Type 外挂模块以及 CSS 的珍贵经验，她们的作品可以说是我学习的跳板，让我能够站在巨人的肩膀上看世界。

RiO 、 REIKO 、 melissa 与 Barabas 在我生活最艰困的时候伸出了援手，使得这本书的撰写无后顾之忧。

本书最后终于得以出版纸本，还得感谢旗标的黄昕暐先生识材而积极联络，也得感谢 lman 提供了许多跟实际出版有关的经验。

最后我还要感谢 macpaul 热情赞助，让我能够更换笔记型计算机，以完成本书的后半段撰写工作。

没有这些人的贡献，这份文件就不可能出现在你的面前；若有人从这份文件受益了，也请把功劳归于她们，我祇不过是喜欢花时间写东西的人罢了。
